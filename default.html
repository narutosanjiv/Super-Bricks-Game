Super Bricks

Level: 1/2 - The everybody can do
Score: 1200

Instructions

  • Use the left and right key arrows and don't let the ball fall!
  • Use ENTER to start the game.
  • Use P to pause en ENTER to start again.

This game is a result of me wanting to play with some HTML5 canvas feature. If you want to add more features and make the code better just fork it on Github.
